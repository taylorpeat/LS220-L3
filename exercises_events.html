Start by adding a submit event to the form. Remember to receive the event argument in order to prevent the form from submitting. Within the callback, read in the character that was entered in to the text input. In order to use it to monitor for a keypress, you will need to convert it to a character code. To do that, we're going to use a little bit of built-in Javascript to convert it. If you've set your input's value to a variable named key, for example, you can call key.charCodeAt(0) to get the character code that represents that character. That number will be used to determine whether or not the right key was pressed in our keypress event.
